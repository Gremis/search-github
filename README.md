## Instalação

## 🏁 Para rodar o projeto:

Clone este repositório em sua máquina:

```bash
$ git clone https://github.com/Gremis/search-github.git
```

cd `search-github` e rode:

```bash
npm install
```

para iniciar:

```bash
npm run start
```
